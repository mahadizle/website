# Technoplus Master Mind Group Website

A modern, responsive landing page for Technoplus Master Mind Group. Built with plain HTML, CSS, and JavaScript so it can be hosted on GitHub Pages with no build step.

## Customize

- Update the bias-focused messaging, impact numbers, and funding links in index.html.
- Replace placeholder contact details with your official info.
- Wire the newsletter and contact forms to your preferred services.

## GitHub Pages deployment

1. Commit the files to a GitHub repository.
2. In repository settings, enable GitHub Pages for the main branch and root folder.
3. Wait for Pages to publish, then use the provided URL.

## Local preview

Open index.html directly in a browser, or serve the folder with a simple static server.
